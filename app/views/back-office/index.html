Show reason for revoked or ceased RUBY-342

Users

  • Need to see the reason for revoke or cease
  • It's mostly area officers that are interested in the reason
  • "Tiny proportion" of exemptions have this - eg <1%
  • Maximum 500 characters, usually less
  • Previous version used <title> to provide a 'hover' view

Design constraints

  • Ideally, use GDS Elements code - don't add custom code.
  • Allow display of up to 500 characters but optimise for about 190.
  • Must be accessible

Sketching

  • Always display reason in exemption column
  • Always display reason below row
  • Details in action column
  • Details in exemption column
  • Details below
  • Add to title tag on status
  • Original - no reason displayed
